# Deep Spark

Source for [stoleckipawel.dev](https://stoleckipawel.dev) — a personal blog on real-time rendering, GPU programming, and performance analysis.

Built with [Hugo](https://gohugo.io) + [Blowfish](https://blowfish.page).

## Quick start

Requires **Hugo Extended** and **Git**.

```bash
hugo server
```

Then open http://localhost:1313.

## Project structure

```
content/           # Markdown articles & pages
config/_default/   # Hugo / Blowfish configuration
layouts/           # Template & shortcode overrides
assets/css/        # Custom styles & colour scheme
static/images/     # Images served as-is
```
